Features

About Project
Terminal is a portfolio project developed for Holberton school. It focuses on analyzing GitHub data and visualizing it in simple understandable format. It involves use of GitHub's API calls, and technologies such as HTML5, CSS, and JavaScript. It uses d3, a JavaScript library, to visualize data acquired after parsing it, to make it more presentable inform of visual graphs and charts.
